============
readme SDOSS
============
X.bonnin, 10-JAN-2012

This folder contains all the files required to run the SDOSS code:
		
		- /archives		contains the old software version.
		- /products 	used to saved output files.
		- /src 			contains the source files of the code. 
		- /tools        used to stored any routine useful for the code.

To launch the code using jsoc data downloading capability, 
run sdo_ss_launcher.pro routine (NOTE: SSW with sdo, vso, and ontology must be loaded before launching the program). A batch file sdo_ss_launcher.batch can also be used to run the software using a script file.
For more details about the files produced, see HELIO_Feature_Description_SunSpots.pdf.
