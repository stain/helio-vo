Helio IDL-Client Install:

If you have a working Java 1.6 installation, skip step 1.

1. Download and Install Java 1.6 from http://www.java.com

2. Download the helio-idlclient-startup.war and helioidl.pro from Hudson

3. Start the java idl-servlet with the command: java -jar helio-idlclient-startup.war

4. Test if the servlet is working correct with the url: http://localhost:8085/AsyncQueryServiceServlet
It should show a idl command starting with "str = {message..."

5. Start idlssw and run helioidl.pro

Catalog Request:
For a list of all catalgs use the command:
result = helioidlshowcatalog()
For more information about a catalog use the command:
result = helioidlshowcatalog(catalog = 'goes_sxr_flare')

HelioQuery:
For a single starttime and endtime use the command:
result = helioidlquery(service = 'ils', starttime = '2003-03-04T00:00:00', endtime = '2003-03-06T00:00:00', from = 'trajectories')
For more than one starttime and endtime use the command:
result = helioidlquery(service = 'ils', starttime = '2003-03-04T00:00:00,2005-03-04T00:00:00', endtime = '2003-03-06T00:00:00,2005-03-06T00:00:00', from = 'trajectories')
For optinal where parameter use the command:
result = helioidlquery(service = 'ils', starttime = '2003-03-04T00:00:00', endtime = '2003-03-06T00:00:00', from = 'trajectories', where = 'wherestatement')
