Helio IDL-Client Install:

If you have a working Java 1.6 installation, skip step 1.

1. Download and install Java 1.6 from http://www.java.com

2. Download helio-idlclient-startup and helio-idlclient-consumer from Hudson

3. Start the java idl-servlet with the command:
   java -jar helio-idlclient-startup.war

4. Test if the servlet is working correct with the url:
   http://localhost:8085/AsyncQueryServiceServlet
   It should show a idl command starting with "function StackTraceElement"

5. Start idlssw

6. edit the following lines in heliovo__define.pro to use the local webserver:
   DEFSYSV, '!heliovo_host', 'localhost'
   DEFSYSV, '!heliovo_port', '8085'
   DEFSYSV, '!heliovo_context', ''

7. create a heliovo-object and print help:
   o=heliovo()
   o->help
